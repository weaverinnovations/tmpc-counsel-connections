# TMPC Counsel Connections

TMCP (Texas Minority Counsel Program) Counsel Connections — tools and data for managing the counsel connections interview program.

## Sample Data Structure

```
data/
├── 01-program-info/        # Program overview and main forms
├── 02-registration/        # Registration forms
├── 03-availability-selection/  # Availability forms, attorney breakdowns, firm selections
├── 04-assignments/         # Interview assignment spreadsheets
├── 05-communications/      # Email templates, FAQs for outside counsel
└── 06-schedules/           # Day-of schedules, FAQs for in-house counsel
```

## Data Source

Sample data from the 2025 TMCP Counsel Connections program (OneDrive export, March 2026).
